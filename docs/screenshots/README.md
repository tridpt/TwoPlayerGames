# 📸 Ảnh chụp màn hình

Thư mục này chứa ảnh dùng trong `README.md`. Bạn tự chụp rồi bỏ vào đây.

## Cách chụp nhanh

1. Chạy app: `npm start` → mở `http://localhost:8777`.
2. Chụp 2 ảnh "ăn tiền" nhất:
   - **menu.png** — màn hình menu chọn game (cuộn lên đầu, thấy sidebar thể loại + lưới game).
   - **gameplay.png** — đang chơi một game đẹp (gợi ý: Cờ Lật, Đẩy Thùng Đôi, Gỡ Bom, hoặc Cờ Vua Mini).
3. (Tùy chọn) **online.png** — sảnh online / phòng có mã, để khoe tính năng online.

## Mẹo cho ảnh đẹp

- Chụp ở cửa sổ rộng ~1200px để giao diện cân đối; hoặc thêm 1 ảnh mobile (DevTools → Ctrl+Shift+M).
- Nén ảnh (PNG hoặc WebP) cho nhẹ; lý tưởng < 300KB mỗi ảnh.
- Muốn ảnh động: quay GIF/WebM ngắn (5–8s) cảnh một nước đi hoặc lật quân.

## Bật ảnh trong README

Sau khi có file, mở `README.md` và **bỏ comment** khối bảng ở mục "📸 Ảnh chụp màn hình"
(đổi tên file cho khớp nếu cần):

```markdown
| Menu chọn game | Đang chơi |
|:---:|:---:|
| ![Menu](docs/screenshots/menu.png) | ![Gameplay](docs/screenshots/gameplay.png) |
```
